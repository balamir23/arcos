# ArcOS

The Desktop Experience for Arc Network.

ArcOS is an open source desktop environment designed for the Arc ecosystem.

Instead of using multiple websites and developer tools ArcOS brings everything into one desktop operating system experience.

---

# Features

- Wallet
- Token Studio
- NFT Studio
- Contract Studio
- Bridge
- Arc Name Service
- Portfolio
- Explorer
- Terminal
- Prediction Arena
- Developer Center
- AI Assistant
- App Store

---

# Vision

ArcOS aims to become the complete operating system for Arc developers creators and users.

Everything should be accessible from one interface.
