# ArcOS

## Vision

ArcOS is a desktop operating system experience built for the Arc Network ecosystem.

The goal is to provide every essential Arc tool inside one interface.

Users should be able to manage wallets deploy contracts create NFTs bridge assets trade on prediction markets and explore the network without leaving ArcOS.
